
# SoulSync using React

SoulSync is a website which shows some classes data dynamically .


## Demo

https://soul-sync-client.web.app/


## Packeges used:

 - react-router-dom
 - firebase
 - tailwind
 - react-icons
 - daisyUi
 - react-toastify
 - sweetalert2
 - axios
 - react-spinners
 - swiper
 - Tan-Stack query
 - Stripe card payment


## Project Feature

- Firebae login register Authentication system
- Private routing systym
- Dynamic data load
- Single Page Application
- Category based item display
- Payment system
- Admin Dashboard
- Instructor Dashboard
- Student Dashboard
- jwt verified
